# bing usage
requirements: 
- `nodejs >= 18`
- bing cookie from bing.com, the cookie name is _U

![image](https://i.imgur.com/bfwkm9z.png)

```shell
$ export BING_COOKIE=<your bing cookie>
$ node index.mjs 你好，西红柿炒鸡蛋咋做
```
